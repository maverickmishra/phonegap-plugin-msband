  • Sensors

    This is where you can start and stop sensors




    • Band Contact

      Provides a way to let the developer know if someone is currently wearing the Band.

    • UV Level

      Provides the current ultra-violet radiation exposure intensity.

    • Skin Temperature

      Provides the current skin temperature of the wearer in degrees Celius (°c)

    • Pedometer

      Provides the total number of steps the wearer has taken.

    • Distance

      Provides the total distance in centimeters, current speed in centimeters per second (cm/s), current pace in milliseconds per meter (ms/m), and the current pedometer mode (such as walking or running).

    • HeartRate

      Provides the number of beats per minute, also indicates if the heart rate sensor is fully locked onto the wearer's heart rate.

    • Accelerometer

      Provides X, Y, and Z acceleration in g units. 1 g = 9.81 meters per second squared (m/s2).

    • Gyroscope

      Provides X, Y, and Z angular velocity in degrees per second (°/sec) units.

  • Tiles
    This section demonstrates how to create and manage tiles on the band.


    • Tile space info

      See how many open tile slots are available.

    • Create a Tile

      Create a tile titled 'PhoneGap' with a guid using icon assets from the app package. This call will fail if a tile already exists with the guid, or if there are no empty tile slots.

    • Remove a Tile

      Remove the tile titled 'PhoneGap' by it's guid. This call will return an error if the tile does not exist.

      Send a message to our tile.
